The Git Commands to Link the Local and Remote Repos

Step 1: Create a new directory.

mkdir (directory name)

This command creates a new directory name (example:mkdir git-project)

Step 2:Navigate to the project directory.

cd /path/to/your/project-directory

This command is used for to navigate to the project

example: cd ~/documents/git-project

Step 3. Initialize a local repository(if not already intilized)

git init

This command is used to Initialize a local Git repository.

Step 4: Create a new file

touch filename.txt

This command is used for creates a new file

Step 5: Check the status of your repository

git status

This command is used to check the status of your repository

Step 6: Add the file to the staging area

git add (filename) or git add .

This command is used for Add the file to the staging area

Step 7: Make a Local commit

git commit -m "intial commit"

This command is used for creates a local commit

Step 8: Add the remote repository URL

git remote add origin <remote-repo-URL>

This command used for Add the remote repository URL

Step 9. Verify the remote repository

git remote -v

This command is used for verify the remote repository

Step 10. Push local code to the remote repository

git push -u origin master

This command is used for to Push the local code to the remote repository
